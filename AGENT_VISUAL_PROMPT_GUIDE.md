# 视觉模块输入 Prompt 指南

本指南供 agent 在运行 `main.py` 后，需要向终端自动输入 VL 检测 prompt 时使用。

## 启用方式

在 `config.py` 中设置：

```python
VL_PROMPT_SOURCE = "terminal"
```

运行：

```bash
python main.py
```

程序出现 `prompt>` 后，输入一条检测 prompt 并回车。若 `VL_PROMPT_SOURCE = "fixed"`，程序会使用代码内置 prompt，不需要终端输入。

## 输出格式要求

prompt 必须要求模型只输出检测框，每个目标一行：

```text
物品名称 [xmin, ymin, xmax, ymax]
```

约束：

- 坐标必须是 0 到 1000 的归一化坐标。
- 每个物品单独一行。
- 不要输出解释、编号列表、Markdown、置信度或多余文字。
- 同类多个物品要分开标注，不要合并成一个框。
- 物品名称要稳定、简短，便于在结果文件中识别，例如 `红色试管_1`、`培养皿_2`。

## 数量限制

如果任务只需要同类物品中的指定数量，为避免误识别，应明确要求只保留面积最大的前 N 个同类目标。

示例：

```text
请检测图中面积最大的 2 个透明试管。只输出检测框，每行格式为：
物品名称 [xmin, ymin, xmax, ymax]
坐标使用 0 到 1000 的归一化坐标。不要输出任何其他文字。
```

## 常用模板

查找单类物品：

```text
请检测图中的所有{物品名称}。只输出检测框，每行格式为：
物品名称 [xmin, ymin, xmax, ymax]
坐标使用 0 到 1000 的归一化坐标。每个物品单独一行，不要输出任何其他文字。
```

查找指定数量：

```text
请检测图中面积最大的{数量}个{物品名称}。只输出检测框，每行格式为：
物品名称 [xmin, ymin, xmax, ymax]
坐标使用 0 到 1000 的归一化坐标。每个物品单独一行，不要输出任何其他文字。
```

查找多类物品：

```text
请检测图中的{物品A}、{物品B}和{物品C}。只输出检测框，每行格式为：
物品名称 [xmin, ymin, xmax, ymax]
坐标使用 0 到 1000 的归一化坐标。每个物品单独一行，不要输出任何其他文字。
```
