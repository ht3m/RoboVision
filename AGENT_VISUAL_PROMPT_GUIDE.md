# 视觉模块输入 Prompt 指南

本指南供 agent 在运行 `main.py` 后，向终端自动输入视觉检测 prompt 时使用。

## 启用终端 Prompt

在 `config.py` 中设置：

```python
VL_PROMPT_SOURCE = "terminal"
```

运行：

```bash
python main.py
```

程序出现 `prompt>` 后，agent 输入 prompt。输入完成后，再输入一个空行结束。

如果 `VL_PROMPT_SOURCE = "fixed"`，程序使用 `Core/detect2sam.py` 中的内置 prompt，不需要终端输入。

## 输入格式

推荐输入多行 Python 字符串片段，格式仿照 `Core/detect2sam.py` 的内置 prompt：

```python
"请为图片中每一个透明试管、透明瓶子和培养皿标注矩形框坐标。\n\n"
"要求：\n"
"- 使用 0 到 1000 的归一化坐标（千分比）\n"
"- 格式：物品名称 [xmin, ymin, xmax, ymax]（不要冒号）\n"
"- 每个物品单独一行\n"
"- 不要添加任何其他描述文字\n\n"
"示例输出格式：\n"
"透明试剂瓶 [200, 150, 400, 500]\n"
"培养皿 [500, 600, 800, 900]"
```

agent 应逐行输入上述字符串片段。程序会自动把这些字符串片段拼接成真正的 prompt，并解析其中的 `\n` 换行。

## 输出约束

prompt 必须要求视觉模型只输出检测框，每个目标一行：

```text
物品名称 [xmin, ymin, xmax, ymax]
```

坐标要求：

- 使用 0 到 1000 的归一化坐标。
- `xmin` 是目标框左边界。
- `ymin` 是目标框上边界。
- `xmax` 是目标框右边界。
- `ymax` 是目标框下边界。

文本要求：

- 不要冒号。
- 不要 Markdown。
- 不要解释文字。
- 不要输出置信度。
- 不要把多个同类物品合并成一个框。
- 每个物品单独一行。

## 物品和数量

如果任务没有数量限制，要求模型检测所有符合条件的目标。

如果任务有数量限制，为了减少误识别，必须要求模型只输出面积最大的前 N 个同类物品。

示例：

```python
"请为图片中面积最大的 2 个透明试管标注矩形框坐标。\n\n"
"要求：\n"
"- 使用 0 到 1000 的归一化坐标（千分比）\n"
"- 格式：物品名称 [xmin, ymin, xmax, ymax]（不要冒号）\n"
"- 每个物品单独一行\n"
"- 不要添加任何其他描述文字\n\n"
"示例输出格式：\n"
"透明试管_1 [200, 150, 400, 500]\n"
"透明试管_2 [500, 600, 800, 900]"
```

## 常用模板

检测所有指定物品：

```python
"请为图片中每一个{物品名称}标注矩形框坐标。\n\n"
"要求：\n"
"- 使用 0 到 1000 的归一化坐标（千分比）\n"
"- 格式：物品名称 [xmin, ymin, xmax, ymax]（不要冒号）\n"
"- 每个物品单独一行\n"
"- 不要添加任何其他描述文字\n\n"
"示例输出格式：\n"
"{物品名称}_1 [200, 150, 400, 500]\n"
"{物品名称}_2 [500, 600, 800, 900]"
```

检测指定数量：

```python
"请为图片中面积最大的 {数量} 个{物品名称}标注矩形框坐标。\n\n"
"要求：\n"
"- 使用 0 到 1000 的归一化坐标（千分比）\n"
"- 格式：物品名称 [xmin, ymin, xmax, ymax]（不要冒号）\n"
"- 每个物品单独一行\n"
"- 不要添加任何其他描述文字\n\n"
"示例输出格式：\n"
"{物品名称}_1 [200, 150, 400, 500]\n"
"{物品名称}_2 [500, 600, 800, 900]"
```

检测多类物品：

```python
"请为图片中每一个{物品A}、{物品B}和{物品C}标注矩形框坐标。\n\n"
"要求：\n"
"- 使用 0 到 1000 的归一化坐标（千分比）\n"
"- 格式：物品名称 [xmin, ymin, xmax, ymax]（不要冒号）\n"
"- 每个物品单独一行\n"
"- 不要添加任何其他描述文字\n\n"
"示例输出格式：\n"
"{物品A}_1 [200, 150, 400, 500]\n"
"{物品B}_1 [500, 600, 800, 900]"
```

## Agent 输入流程

1. 运行 `python main.py`。
2. 等待终端出现 `prompt>`。
3. 按本指南输入多行 Python 字符串片段。
4. 输入一个空行结束 prompt。
5. 等待视觉流程完成。
6. 读取 `result/vision_result.txt` 获取基坐标结果。
