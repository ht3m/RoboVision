# 推荐输入格式

agent 在 `prompt>` 后推荐输入多行 Python 字符串片段，格式仿照 `Core/detect2sam.py` 中的内置 prompt。每一行都是一个字符串字面量，最后输入一个空行结束：

```python
"请为图片中每一个透明试管、透明瓶子和培养皿标注矩形框坐标。\n\n"
"要求：\n"
"- 使用 0 到 1000 的归一化坐标（千分比）\n"
"- 格式：物品名称 [xmin, ymin, xmax, ymax]（不要冒号）\n"
"- 每个物品单独一行\n"
"- 不要添加任何其他描述文字\n\n"
"示例输出格式：\n"
"透明试剂瓶 [200, 150, 400, 500]\n"
"培养皿 [500, 600, 800, 900]"
```

如果要指定物品和数量，把第一行改成目标任务；有数量限制时，必须要求“面积最大的 N 个”，例如：

```python
"请为图片中面积最大的 2 个透明试管标注矩形框坐标。\n\n"
"要求：\n"
"- 使用 0 到 1000 的归一化坐标（千分比）\n"
"- 格式：物品名称 [xmin, ymin, xmax, ymax]（不要冒号）\n"
"- 每个物品单独一行\n"
"- 不要添加任何其他描述文字\n\n"
"示例输出格式：\n"
"透明试管_1 [200, 150, 400, 500]\n"
"透明试管_2 [500, 600, 800, 900]"
```

# 视觉模块输入 Prompt 指南

本指南供 agent 在运行 `main.py` 后，需要向终端自动输入 VL 检测 prompt 时使用。

## 启用方式

在 `config.py` 中设置：

```python
VL_PROMPT_SOURCE = "terminal"
```

运行：

```bash
python main.py
```

程序出现 `prompt>` 后，输入一条检测 prompt 并回车。若 `VL_PROMPT_SOURCE = "fixed"`，程序会使用代码内置 prompt，不需要终端输入。

## 输出格式要求

prompt 必须要求模型只输出检测框，每个目标一行：

```text
物品名称 [xmin, ymin, xmax, ymax]
```

约束：

- 坐标必须是 0 到 1000 的归一化坐标。
- 每个物品单独一行。
- 不要输出解释、编号列表、Markdown、置信度或多余文字。
- 同类多个物品要分开标注，不要合并成一个框。
- 物品名称要稳定、简短，便于在结果文件中识别，例如 `红色试管_1`、`培养皿_2`。

## 数量限制

如果任务只需要同类物品中的指定数量，为避免误识别，应明确要求只保留面积最大的前 N 个同类目标。

示例：

```text
请检测图中面积最大的 2 个透明试管。只输出检测框，每行格式为：
物品名称 [xmin, ymin, xmax, ymax]
坐标使用 0 到 1000 的归一化坐标。不要输出任何其他文字。
```

## 常用模板

查找单类物品：

```text
请检测图中的所有{物品名称}。只输出检测框，每行格式为：
物品名称 [xmin, ymin, xmax, ymax]
坐标使用 0 到 1000 的归一化坐标。每个物品单独一行，不要输出任何其他文字。
```

查找指定数量：

```text
请检测图中面积最大的{数量}个{物品名称}。只输出检测框，每行格式为：
物品名称 [xmin, ymin, xmax, ymax]
坐标使用 0 到 1000 的归一化坐标。每个物品单独一行，不要输出任何其他文字。
```

查找多类物品：

```text
请检测图中的{物品A}、{物品B}和{物品C}。只输出检测框，每行格式为：
物品名称 [xmin, ymin, xmax, ymax]
坐标使用 0 到 1000 的归一化坐标。每个物品单独一行，不要输出任何其他文字。
```
