# 视觉模块输出理解指南

视觉模块每次运行 `main.py` 时都会覆写：

```text
result/vision_result.txt
```

## 文件格式

每个成功定位到机器人基坐标的物品占一行：

```text
[物品名字及编号]：[x, y, z]
```

示例：

```text
[透明试管_1]：[0.1234, -0.0567, 0.2450]
[培养皿_2]：[0.1800, -0.0321, 0.1988]
```

## 含义

- 方括号左侧内容是视觉检测得到的物品名称及编号。
- 方括号右侧 `[x, y, z]` 是机器人基坐标系下的三维坐标。
- 坐标单位为米。
- 文件只包含最终可用于机器人动作规划的基坐标，不包含相机坐标、像素框或中间调试信息。

## 空文件

如果文件为空，表示本次运行没有可用的最终基坐标。常见原因：

- VL 没有检测到目标。
- SAM 或点云重建没有生成有效物体点云。
- 机械臂位姿读取失败，无法转换到基坐标系。

读取时应把空文件当作“没有可执行目标”，不要沿用上一次运行的结果。
